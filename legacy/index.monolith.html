Horas mensais

⊞
Arraste o CSV ou clique para selecionar
Google Sheets → File → Download → CSV

A planilha deve ter as colunas MÊS, SEMANA, HORAS/MINUTOS, SISTEMA, CATEGORIA e DESCRIÇÃO.
Formatos de tempo aceitos: 1:30:00 ou 1:30.

● Conectado
Escolha o método:
Como gerar a URL:
1. Abra sua planilha no Google Sheets
2. File → Share → Publish to web
3. Em "Link", selecione a aba da planilha (ex: Fevereiro)
4. Mude o formato para Comma-separated values (.csv)
5. Clique em Publish e copie a URL gerada
Como configurar:
1. Abra sua planilha → Extensions → Apps Script
2. Apague o código existente e cole o código abaixo
3. Deploy → New deployment → Web app
4. Execute as: Me · Who has access: Anyone
5. Clique em Deploy, autorize e copie a URL 
function doGet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sh = ss.getActiveSheet();
  const data = sh.getDataRange().getValues();
  const headers = data[0].map(h => String(h).trim());
  const rows = data.slice(1)
    .filter(r => r.some(c => c !== ''))
    .map(r => {
      const obj = {};
      headers.forEach((h, i) => obj[h] = r[i]);
      return obj;
    });
  const out = ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
  return out;
}
